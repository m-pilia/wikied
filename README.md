wikied - semi-automated wiki editor
===================================

This program is a semi-automated editor which can operate on the pages of a [MediaWiki](https://www.mediawiki.org/wiki/MediaWiki) site. It is implemented in [python](https://www.python.org) 3 and
[Qt](http://www.qt.io) 5 ([pyqt](https://www.riverbankcomputing.com/software/pyqt/intro)).

The project is a work in progress, and the current version has main basic features working already.
